# 信息来源可信度评估指南

## 来源分级

### A 级（高可信）
- 同行评审学术论文（Google Scholar、PubMed）
- 政府官方网站（.gov, .edu）
- 国际权威机构（WHO, ISO, IEEE）
- 企业官方文档和公告

### B 级（中可信）
- 知名科技媒体（TechCrunch, Wired, Ars Technica）
- 专业行业报告（Gartner, Forrester）
- 知名技术博客（有作者署名）

### C 级（低可信，需多源验证）
- 论坛帖子（Reddit, Stack Overflow 回答需验证）
- 个人博客（无明确资质）
- 维基百科（可作为起点，但需追溯来源）
- 社交媒体内容

## 评估要点

1. **作者资质**：是否有相关领域专业背景？
2. **发布时间**：技术类内容是否过时（> 2年）？
3. **引用情况**：是否引用了可核实的一手来源？
4. **利益冲突**：发布者是否有商业动机？
5. **一致性**：结论与其他来源是否一致？
